# Cognitive Friction

Cognitive friction is the hidden tax on modern knowledge work.

It appears as:

- context switching
- repeated re-explanation
- fragmented documents
- forgotten rationale
- shallow summaries
- invisible tradeoffs
- disagreement without synthesis
- decisions without memory

Most productivity software treats this as a speed problem.

This project treats it as an infrastructure problem.

The goal is not to remove all friction. Some friction is where good thinking happens. The goal is to remove the wrong friction: the friction of lost context, missing reasoning, and untraceable decisions.
