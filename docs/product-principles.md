# Product Principles

1. Cognitive depth before productivity theater.
2. Reasoning transparency before polished prose.
3. Comparison before conclusion.
4. Memory as evolving context, not archival storage.
5. Calm interfaces for serious thinking.
6. Fewer workflows, deeper loops.
7. Human agency stays visible.

Avoid:

- chatbot-first UX
- generic AI assistant flows
- flashy demos
- excessive multi-agent orchestration
- automation that hides reasoning

Optimize for:

- synthesis quality
- strategic clarity
- uncertainty handling
- tradeoff visibility
- continuity across sessions
