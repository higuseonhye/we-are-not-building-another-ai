# Manifesto

Most AI tools generate answers.

We are building systems that help humans think.

The next frontier is not another assistant, another chat surface, or another document summarizer. The frontier is cognitive workflow infrastructure: systems that preserve reasoning, expose assumptions, compare perspectives, and help ideas evolve over time.

Modern knowledge work is not blocked by lack of information. It is blocked by fragmented context, shallow synthesis, invisible tradeoffs, forgotten reasoning, and decision fatigue.

This repository exists to prototype a different posture:

- AI should augment thought processes, not just produce outputs.
- Synthesis should preserve uncertainty instead of hiding it.
- Memory should store evolving reasoning, not static summaries.
- Comparison should make disagreement useful.
- Workflows should help people reclaim cognitive clarity and time for meaningful life.

We are not building another AI wrapper.

We are building cognitive workflow infrastructure for the AI era.
